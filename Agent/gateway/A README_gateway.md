# gateway — 对外入口

Agent 与外部世界交互的入口层，提供两种接入方式。

## 文件说明

| 文件 | 职责 |
|------|------|
| `cli.py` | 终端交互入口：启动 CLI 会话、处理用户输入 |
| `api.py` | HTTP / WebSocket 入口：供可视化前端接入 |
| `__init__.py` | 包初始化 |

## 两种入口对比

| 入口 | 适用场景 |
|------|----------|
| `cli.py` | 本地开发调试、命令行直接使用 |
| `api.py` | 可视化面板、远程调用、步骤流程监控 |

## 与其他模块的关系

- 两者最终都调用 `core/loop.py` 驱动对话
- `api.py` 通过 `core/events.py` 向前端推送实时事件流
