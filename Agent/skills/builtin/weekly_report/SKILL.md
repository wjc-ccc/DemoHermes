# 周报生成技能

当用户要求写周报、汇总一周工作时，按以下步骤执行：

## 步骤

1. 从用户的描述中提取工作事项；信息不足时，先向用户追问（做了哪些事、有无数据、下周计划）。
2. 将事项按「完成 / 进行中 / 阻塞」分类。
3. 按下方格式输出周报，语言简洁、每条一句话、能量化的尽量带数字。

## 输出格式

```
# 本周工作周报

## 一、已完成
- 事项 + 结果/产出

## 二、进行中
- 事项 + 当前进度（百分比或阶段）

## 三、问题与阻塞
- 问题 + 需要的支持（无则写「无」）

## 四、下周计划
- 计划事项
```

## 注意

- 不要编造用户没提过的工作内容。
- 用户只给了零散记录时，可以合理归并同类项，但不得夸大成果。
